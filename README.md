4. Project Description
The Smart Student Management System will allow administrators to manage student records digitally. The system will provide functionalities such as adding, updating, deleting, and searching student data. It will also generate reports and display a dashboard for quick insights.
5.## Features
- Secure Login System (Email + Password validation)
- Email must contain '@' symbol
- Password must have Uppercase + Number + Special character
- Add New Students
- Edit Existing Student Records
- Delete Students with confirmation
- Search Students by Name, Roll No, Department
- View Complete Student Details
- Dashboard with Total, Male, Female student stats
- Input Validation on all forms
- OOP Concepts used throughout
6. ## Installation Steps
1. Install Java JDK 8 or above from https://java.com
2. Install Eclipse IDE from https://eclipse.org
